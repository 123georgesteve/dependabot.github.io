How it works

Dependabot is here to help keep your Ruby dependencies up to date. It does all the legwork, leaving you with easy to review pull requests for any outdated dependencies. Each day:

Dependabot checks for updates

Every day, Dependabot pulls down your Ruby dependency files and looks for any outdated requirements.

Dependabot opens pull requests

If any of your dependencies are out-of-date, Dependabot opens individual pull requests to bump each one.

You review and merge

You check the changelog and release notes linked to from dependabot's pull requests, and hit merge.
